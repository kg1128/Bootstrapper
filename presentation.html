The Thesis

The Shift

Two eras of AI — a different division of labor

Phase 1 · Helper
AI assists your work.
  • Summarize documents, draft emails
  • Human still owns architecture & logic
  • Value: saves minutes per task
"Summarize this credit memo into key risks and investment implications."

"Help me write code to import and clean this Excel data."
Phase 2 · Agent
AI builds the solution.
  • You define the what — AI owns the how
  • Writes the model, code, UI, exports — end to end
  • Value: speed, cost, new capability
"Recommend a portfolio rotation strategy with a loss budget of $X that maximizes net yield."

"Build a deck summarizing YTD portfolio activities for a client."

What We Built

SAA Optimizer — production-grade

Portfolio Optimization
Construct the optimal portfolio given investment objectives, liability profile, regulatory and internal constraints, and current market conditions.
Sensitivity Analysis
Run multi-parameter grid searches across any constraint combination. How does extending duration by 0.5 years affect optimal yield? What is the yield-vs-credit-quality trade-off? Which allocations can hit the target yield?
Interactive Visualization
Dynamic, fully customizable charts built on demand — allocation breakdowns, efficient frontiers, and cash flow matching visualizations.
Session Management
Save, reload, and compare multiple portfolio scenarios side by side across sessions.
Excel Export
Formatted, multi-tab Excel report with full portfolio summary — ready for internal review or client distribution.

How It Was Built

Director + agent — an iterative loop

1 · Human
Define objective, constraints, and data.
2 · AI Agent
Design architecture, write optimizer and UI.
3 · Human
Review, test, redirect.
4 · Repeat
Features ship in minutes, not weeks.
Interface
Natural-language conversation — no code written by hand.
Cadence
Every feature in this platform was shipped this way.
Example Turn
"Add a constraint on WARF and show the impact on yield as the limit tightens." Agent edits the optimizer, updates the UI with a new slider, regenerates the sensitivity grid, and ships the working version — in a single session.

Business Value

Why it matters

Speed
  • Analytics and reporting that took hours now run in minutes.
  • Faster model development, data work, and deck building.
  • Decisions move faster — analysis on demand, not queued.
Cost
  • Brings capability in-house — less reliance on external tools.
  • Scales output without scaling headcount.
  • Fewer errors, less rework.
Capability
  • Unlocks analysis previously impractical — sensitivity grids, real-time efficient frontiers, cash flow matching at scale.
  • Analysts shift from data gathering to judgment; PMs gain a thinking partner.
06 / 09
Human + AI

The role doesn't disappear — it elevates

Where humans stay in the loop

  • Validation. AI makes mistakes — domain expertise is essential.
  • Compliance & audit. Still requires human oversight.
  • Production deployment. Engineering review is still required.

What changes for the human

  • The role doesn't disappear. It elevates — from doing the work to directing it.
  • Judgment becomes the scarce input. Problem framing and validation matter more, not less.
  • Expertise compounds. Deeper domain knowledge = better agent output.

Summary

Key takeaways

01
AI has crossed a threshold — from assistant to agent. The division of labor has fundamentally shifted. You define the problem; AI builds the solution.
02
The bottleneck is no longer technical. The question has shifted from "can we build it?" to "can we define it well?" — domain expertise is now the scarce and decisive input.
03
The value is concrete. Faster decisions, lower operational cost, and analytical capabilities that were previously out of reach.
04
Domain expertise is now the most powerful lever. Subject-matter experts can operationalize ideas that would never have reached the dev queue.
